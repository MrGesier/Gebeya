# Gebeya PointsFi Simulator

Streamlit app to simulate Gebeya's **PointsFi** framework over 1 year:

- Editable quest / task table (from the Gebeya study)
- User-level simulation of points accrual + inactivity erosion
- Dynamic tiering with African-inspired tiers: **Nile, Baobab, Sankofa, Kilimanjaro**
- Conversion from G-Points â†’ **G-Token**
- Estimation of potential **sell pressure in USD** given token price and daily sell cap
- Score evolution plots and leaderboards
